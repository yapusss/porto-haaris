# 🚀 Premium Portfolio Website

Modern, minimalist portfolio built with **Next.js 14**, **TypeScript**, and **Tailwind CSS**.

## ⚡ Quick Start

```bash
# 1. Install
npm install

# 2. Run dev server
npm run dev
# Open http://localhost:3000
```

## 🚀 Deploy

```bash
# Via Vercel (recommended)
npm i -g vercel && vercel
```

Or push to GitHub and connect to Netlify.

## 💡 Tips

- **Images**: Use Unsplash (free, high quality)
- **Email Form**: Use Formspree.io (easiest)
- **Colors**: Edit `tailwind.config.ts` to change accent color
- **Font**: Edit `globals.css` to change typography
