## 💡 Pro Tips

1. **For images**: Use Unsplash (free, high quality)
   - Aspect ratio: 16:9
   - Size: ~600x400px

2. **For contact form**: Use Formspree.io (easiest)
   - No backend needed
   - 50 submissions/month free

3. **For social links**: Get profile URLs:
   - GitHub: github.com/your-username
   - LinkedIn: linkedin.com/in/your-profile
   - Twitter: twitter.com/your-handle

4. **For deployment**: Use Vercel
   - Free tier includes all you need
   - Custom domain support
   - Auto-deploys on git push
