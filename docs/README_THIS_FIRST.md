## 💡 EDITING TIPS

**For Images:**

- Use Unsplash (free, high quality)
- Aspect ratio: 16:9 (e.g., 600x400)
- Or put in `public/` folder

**For Projects:**

- Add 5-6 best projects
- Real GitHub links
- Real live project links
- Actual tech stack

**For Testing:**

- Test on mobile (DevTools)
- Check all links
- Verify images load
- Use CHECKLIST.md
