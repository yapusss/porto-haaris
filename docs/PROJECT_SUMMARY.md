## 🎨 CUSTOMIZATION OPTIONS

### Colors

Edit `tailwind.config.ts`:

```typescript
accent: {
  cyan: '#00d9ff',   // Change this color
  teal: '#0ea5b8',
  hover: '#06b6d4',
}
```

### Typography

Edit `globals.css`:

```css
@import url("https://fonts.googleapis.com/css2?family=YourFont:wght@400;500;600;700;800");
```

### Spacing/Layout

Edit component files directly or use Tailwind classes.
